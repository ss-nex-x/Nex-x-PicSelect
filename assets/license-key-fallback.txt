Nex-X Pix Select - License Key Fallback Package

This file helps support verify your activation request when email delivery is delayed.

1. Keep this file unchanged.

   - Order ID / payment reference
   - Device name from app activation screen
3. Support will issue or re-send your correct activation key.

Important:
